# **App Name**: ArticleFlow

## Core Features:

- Article Feed: Display a continuous feed of articles in a TikTok-like interface with swipeable horizontal navigation for skipping articles and vertical scrolling to read the article.
- Article Recommendation: AI-powered content suggestion engine which utilizes machine learning and user data to recommend content; the recommendation tool monitors what the user reads or skips, to build a personalized article profile and create more suitable suggestions over time.
- Article Saving: Allow users to save articles for later reading.
- Basic Article Metadata: Ingest the source url, title, author, date published and content into the Cloud SQL database

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to create a modern and focused reading experience.
- Background color: Very light indigo (#F0F2F9) to provide a high-contrast backdrop.
- Accent color: Purple (#9C27B0) to draw attention to key interactive elements, as this creates a vivid contrast without distracting the user from the reading task.
- Body font: 'Inter', a grotesque-style sans-serif font, for readability and a modern look.
- Headline font: 'Space Grotesk' a proportional sans-serif, which will look futuristic and stand out as title.
- Minimalist icons for save, share, and other actions to ensure clarity and ease of use.
- Clean, card-based layout with clear separation between articles to aid browsing and readability.
- Smooth transition animations when swiping between articles to enhance the browsing experience.
